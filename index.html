Follow these 5 steps to connect this survey to Google Sheets. Takes about 5 minutes.

1

Create a Google Sheet

Go to sheets.google.com and create a new blank spreadsheet. Name it something like "Survey Responses Master". Leave it open.

2

Open Apps Script

In your Google Sheet, click Extensions → Apps Script. A new tab will open with a code editor. Delete any existing code in the editor.

3

Paste this script

Copy the entire script below and paste it into the Apps Script editor:

function doPost(e) { try { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); // Write header row if sheet is empty if (sheet.getLastRow() === 0) { sheet.appendRow(data.headers); // Bold the header row sheet.getRange(1, 1, 1, data.headers.length).setFontWeight("bold"); sheet.setFrozenRows(1); } // Append the response row sheet.appendRow(data.values); return ContentService .createTextOutput(JSON.stringify({ status: "success" })) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ status: "error", message: err.toString() })) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { return ContentService .createTextOutput(JSON.stringify({ status: "ok", message: "Survey endpoint active" })) .setMimeType(ContentService.MimeType.JSON); }
4

Deploy as Web App

In the Apps Script editor, click Deploy → New deployment. Set:
• Type: Web app
• Execute as: Me
• Who has access: Anyone
Then click Deploy. Authorise when prompted. Copy the Web App URL that appears.

5

Paste the URL in the Connect tab

Go to the Google Sheets tab above and paste the Web App URL, then click Save & Test. You should see a green "Connected" status. You're done! 🎉
